# Biblioteca Digital Manager

Sistema desenvolvido para auxiliar bibliotecas no gerenciamento de documentos digitais.

## Funcionalidades

O sistema oferece as seguintes funcionalidades:
- Listagem de documentos por tipo
- Listagem de documentos por ano
- Inclusão de documentos no acervo
- Renomeação de documentos
- Remoção de documentos
- Criação e remoção de diretórios

## Tecnologias

- Python 3
- Git
- GitHub

## Execução

```bash
python src/gerenciador.py
```
