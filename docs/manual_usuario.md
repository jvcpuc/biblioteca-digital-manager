# Manual do Usuário

## Objetivo

Gerenciar documentos digitais da biblioteca.

## Menu Principal

1. Listar documentos
2. Adicionar documento
3. Renomear documento
4. Remover documento
5. Criar diretório
6. Remover diretório
0. Sair

## Funcionalidades

### Listar documentos
Exibe os documentos organizados por tipo e ano.

### Adicionar documento
Permite cadastrar um novo documento.

### Renomear documento
Altera o nome de um documento existente.

### Remover documento
Exclui um documento selecionado.

### Criar diretório
Cria uma nova pasta para organização.

### Remover diretório
Remove uma pasta e seu conteúdo.
